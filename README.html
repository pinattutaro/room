Pure CSS 3D Room

HTMLとCSSだけで、インタラクティブな3Dの世界を構築するプロジェクト

What is This?

このプロジェクトは、JavaScriptを一切使わずに、Pure CSSの力だけで3Dのオブジェクトや部屋を作成するためのライブラリ（default.css）です。

CSS変数（--width, --height, --thickなど）をインラインスタイルで指定するだけで、直感的に3Dモデルを組み立てることができます。

Simple Example: Creating a Box

最も基本的な使い方です。.boardクラスを持つdiv要素を作成し、CSS変数でサイズと色を定義します。スタイルは継承されるため、.boardの--width及び--heightは省略できます。

<div class="room">
    <div class="object" style="
    --vertical: -120px;
    --width: 120px;
    --height: 120px;">
        <div class="board" style="
        --width: 120px;
        --height: 120px;
        --thick: 120px;
        --color: #e74c3c;
        --side-color: #c0392b;"></div>
    </div>
</div>

Advanced Example: A Piece of Furniture

複数の.boardや.hollow（穴）を組み合わせることで、より複雑なオブジェクトも作成できます。.objectでグループ化し、入れ子にすることで、家具のようなモデルも構築可能です。

より複雑な完全なシーンは、sample/sam1.htmlで確認できます。
